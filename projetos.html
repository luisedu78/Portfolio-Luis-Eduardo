AutoReport - Gerador de Relatórios (Python)

Script Python que coleta dados de uma planilha Excel e gera relatórios em PDF. Ideal para automações empresariais.

⚠️ Este projeto não é executável no navegador, mas o código está abaixo:

import pandas as pd
from fpdf import FPDF

df = pd.read_excel("dados.xlsx")
pdf = FPDF()
pdf.add_page()
pdf.set_font("Arial", size=12)

for i, row in df.iterrows():
    pdf.cell(200, 10, txt=str(row.to_dict()), ln=True)

pdf.output("relatorio.pdf")

TaskFlow - Lista de Tarefas (JavaScript)

Adicione, remova e marque tarefas. Tudo salvo no navegador usando localStorage.

function addTask() {
  const task = document.getElementById("task").value;
  if(task){
    const li = document.createElement("li");
    li.textContent = task;
    li.onclick = () => li.remove();
    document.getElementById("list").appendChild(li);
    document.getElementById("task").value = "";
  }
}
