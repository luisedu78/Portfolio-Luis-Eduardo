Projeto 1: Conversor de Temperatura (JavaScript)

Converte Celsius para Fahrenheit em tempo real.

function convert() { var c = document.getElementById('celsius').value; var f = (c * 9/5) + 32; document.getElementById('output').textContent = f + ' °F'; }

Projeto 2: Calculadora de IMC (Python)

Script em Python para calcular o IMC com entrada de peso e altura.

# calculadora_imc.py peso = float(input("Digite seu peso (kg): ")) altura = float(input("Digite sua altura (m): ")) imc = peso / (altura ** 2) print(f"Seu IMC é: {imc:.2f}")
